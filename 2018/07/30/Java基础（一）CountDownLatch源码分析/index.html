总结

  学习了Doug Lea大神的源码后，什么函数式编程，面向对象编程，一切都觉过眼云烟，所有的函数方法存在的意义，都是为了重用。你在层层重用面前，从这个方法跳到另外一个方法，当你觉得终于达到了终点的时候，一切其实又回到了原点。
