拥抱Docker（一）常用命令

Posted on 2019-05-27 Edited on 2019-10-25 In Docker

  Docker是Docker.Inc开发的一款开源容器引擎，它在容器的基础上，将文件、网络、进程等一系列资源进行进一步封装和管理，通过对镜像的打包和封装，以及引入Docker Registry集中存储Docker镜像，从而简化了整个开发、测试和部署的环境和流程。

镜像

查找仓库镜像

docker search [镜像仓库]

上面search命令搜索所有镜像仓库中包含nginx的镜像，对应NAME格式用户名/镜像仓库，一般官方仓库仅有仓库名，比如上面的第一个镜像仓库是官方nginx仓库。

创建、运行

容器创建并运行的命令比较复杂，这里只简单地讨论：

docker run [参数] [镜像]

这里列举几个常用的的参数简单说明：

  • -p: 指定端口映射，宿主机端口:容器端口
  • -d: 以后台的形式运行容器
  • -t: 给容器分配一个伪终端，通常与-i配合使用
  • -i: 以交互的形式运行，与-t配合使用
查看

列出所有运行中的容器：

docker ps [参数]

列举几个常用的参数：

  • -a: 显示所有的容器，包括未运行的
  • –format :指定返回的模板

查看容器详情：

docker inspect[容器]
